# 初学者快速入口

## 目标

先完成一个最小闭环：**建项目 → 画电机启停 → 检查 → 出部件表/PDF → 备份**。完成这个闭环后，再进入 PLC、端子、电缆和宏。

资料选择见 [[10_官方资料/初学者精选学习资料|初学者精选学习资料]]。本页只负责任务顺序，不重复教程正文。

## 每次学习的固定节奏

每次控制在 45～60 分钟：

1. **10 分钟**：只读当前任务相关的官方帮助或一个 eLearning 模块；
2. **30～40 分钟**：在同一个练习项目中操作；
3. **5～10 分钟**：保存截图、PDF、检查结果或备份，并记录一个问题。

没有可复核输出，不把任务标为“掌握”。

## 第一轮：七个最小任务

| 次序 | 任务 | 只读资料 | 可验收输出 |
|---:|---|---|---|
| 0 | 确认版本、语言、许可证范围和练习目录；建立虚拟机恢复点 | [[00_Home/EPLAN学习环境|EPLAN 学习环境]] | 环境信息与恢复点记录 |
| 1 | 从可确认来源的基本项目创建练习项目，并建封面、结构概览、原理图页 | [[20_基础知识/项目与页/项目与页面_入门|项目与页面：入门]] | 三页项目和页导航器截图 |
| 2 | 用插入中心放置断路器、接触器、热继电器、电机和按钮；理解符号、功能、主功能 | [[10_官方资料/初学者精选学习资料#EPLAN 2025 最小书签集|2025 最小书签集]] | 对象齐全的未接线原理图 |
| 3 | 设置 DT，完成自动连接和必要的连接符号；观察交叉引用 | 同上：设备标识与自动连接 | 可读的电机启停主回路和控制回路 |
| 4 | 用页、设备和连接导航器反查对象；修正重复 DT、未连接点等基础问题 | 同上：项目检查 | 第一轮检查消息及修正前后记录 |
| 5 | 只给本案例使用的主要设备分配少量真实部件，生成部件表并导出 PDF | 同上：部件管理与报表 | 部件表和 PDF |
| 6 | 关闭并重新打开项目，执行最终检查，生成 `.zw1` 备份并写复盘 | 同上：项目备份 | 项目备份、检查摘要和一页复盘 |

## 第一轮通过标准

- [ ] 项目能够关闭并重新打开；
- [ ] 页面名称、页面类型和描述可在页导航器中复核；
- [ ] 每个主要设备有合理且不重复的 DT；
- [ ] 连接不是只靠自由图形线伪装；
- [ ] 能从检查消息跳转到对象并完成至少一次修正；
- [ ] 部件表来自项目数据，而不是手工抄表；
- [ ] 已导出 PDF，并有可恢复的 `.zw1` 备份；
- [ ] 已记录当前版本、使用的基本项目和仍未解决的问题。

## 第二轮再学什么

按 [[00_Home/学习路线图|学习路线图]] 继续：

1. PLC 盒、连接点、I/O 地址；
2. 端子排、插头、多芯电缆和屏蔽；
3. 设备宏、占位符和值集；
4. 端子图表、电缆图表、PLC 图表和连接列表；
5. 项目检查方案、模板和公司标准候选。

安全回路、急停和 STO 只做文档表达练习时，也必须注明依据、适用边界和待专业复核项，不能把练习图当作已验证安全设计。

## 卡住时怎么查

1. 在当前对话框或消息上按 `F1`；
2. 到 [[10_官方资料/官方帮助索引|官方帮助索引]] 查对象名；
3. 再查 [[10_官方资料/初学者精选学习资料|精选资料]]中的旧版伴读；
4. 仍未解决时，用 [[99_Templates/Template_问题记录|问题记录模板]] 保存现象、环境、预期与排查过程。
