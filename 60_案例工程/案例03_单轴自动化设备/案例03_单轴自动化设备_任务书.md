# 案例 03：单轴伺服自动化设备电气工程任务书

本任务书只定义学习案例的边界、输入和验收证据，不预设未经确认的器件型号、地址、接线或安全等级。案例基线来自 [[README#8. 首个综合案例|Vault 根文件第 8 章]]。

## 学习目标

在 EPLAN Electric P8 2025 中完成一套小型单轴伺服自动化设备的电气工程，并从原理图数据生成可检查的端子、电缆、部件和连接资料。

## 建议范围

- 三相或单相总电源
- 总断路器、24 VDC 电源及支路保护
- PLC 或工业控制器
- 伺服驱动器和伺服电机
- 急停、STO、正负限位和原点传感器
- 启动、停止、复位和状态指示
- 柜内端子排
- 电机、编码器、传感器和操作盒电缆
- 一个多芯电缆案例
- 部件表、端子图表、电缆图表和连接列表

> [!warning] 安全边界
> 急停、STO 和其他安全相关内容目前仅为文档学习范围。实际架构、性能等级、器件选择和验证方法必须依据适用标准、风险评估及专业复核后确定。

## 完成顺序

- [ ] 定义设备边界和 I/O 清单
- [ ] 建立项目结构、页结构和设备标识规则
- [ ] 完成主回路和 24 VDC 配电
- [ ] 完成控制、安全和伺服信号的文档表达
- [ ] 建立 PLC I/O 和地址
- [ ] 建立端子、插头和多芯电缆
- [ ] 分配已核实的真实部件
- [ ] 建立可复用设备宏
- [ ] 自动生成报表
- [ ] 执行检查、修正并归档
- [ ] 提取公司模板和标准回路候选

## 最小首轮实践

在 30 分钟内只完成以下工作：

1. 新建练习项目，不使用公司生产项目或主数据。
2. 记录所选项目类型和创建路径。
3. 建立封面、结构概览和一张原理图页。
4. 导出或截图页导航器结果，作为阶段一完成证据。
5. 将操作路径、结果和问题整理为专题或问题笔记。
